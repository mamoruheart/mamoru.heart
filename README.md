# mamoru.heart

Mamoru Heart's personal website | 南出さんの個人サイト

## Prerequisites

```yaml
node.js: ^18.20.7
yarn: 1.22.22

next.js: 14.1.4
tailwindcss: 3.4.3
```

## Getting Started

```bash
# install dependencies
$ yarn install

# start dev server
$ yarn dev
```

### Local server is running at http://localhost:3000/
